## 🚀 Featured Projects

### 🏋️ [HealthFit — AI-Powered Health & Fitness Tracker](https://github.com/shikharY115/HealthFit-AI-Powered-Health-Fitness-Tracker)
Full-stack health and fitness platform with an AI-powered meal-logging assistant, built as part of a team.
- Built secure backend APIs (Node.js, Express, MongoDB) for authentication, nutrition tracking, BMI monitoring, and workout management
- Integrated Gemini 1.5 Flash to turn natural-language meal descriptions into structured nutrition data, with a fallback parser to keep logging working if the AI service was unavailable
- Connected the USDA FoodData Central API to MongoDB data models for consistent calorie and macronutrient tracking
- Added Google Fit OAuth for automated step syncing and built the analytics dashboard with Chart.js
- Collaborated with teammates through pull-request reviews to keep frontend and backend data contracts aligned

`React` `Node.js` `Express.js` `MongoDB` `Google Gemini`

### 🏠 [Online Hospitality Marketplace](https://github.com/shikharY115/Online_Hospitality_Marketplace)
Full-stack rental marketplace for listing and booking properties in real time.
- Architected and deployed a rental platform supporting multiple property listings with secure image uploads
- Built authentication and authorization with Passport.js (local strategy) and MongoDB-backed sessions, supporting 1,000+ users
- Implemented scalable CRUD operations so hosts can add, edit, and monitor listings with real-time availability tracking
- Reduced booking management time by 40% through a streamlined listing and booking flow

`JavaScript` `Node.js` `Express.js` `MongoDB`

### 🗺️ [Pix2Pix Aerial Image-to-Map Generator](https://github.com/shikharY115/pix2pix-aerial-image-to-map-generator)
Conditional GAN that translates satellite imagery into map-style renderings.
- Built a Pix2Pix image-to-image translation pipeline in PyTorch with a U-Net generator and PatchGAN discriminator
- Combined adversarial, pixel-reconstruction, and edge-preservation losses to improve structural accuracy and fine detail in generated maps
- Added checkpointing to resume interrupted training runs and data augmentation to improve generalization on a limited paired dataset
- Deployed the trained generator in a Streamlit app where users upload satellite imagery and download the generated map

`PyTorch` `Streamlit` `TorchVision`

### 🧠 [Brain Tumor Detection & Segmentation](https://github.com/shikharY115/brain-tumor-detection-segmentation)
Deep learning pipeline for classifying and segmenting brain tumors from MRI scans.
- Built a preprocessing pipeline (normalization, augmentation, mask cleaning) to handle TIFF-format MRI data
- Applied transfer learning with VGG16 for classification and a custom Res-U-Net for pixel-level tumor segmentation
- Achieved 98% classification accuracy and a 0.88 IoU segmentation score
- Built an interactive Streamlit app for uploading MRI images and viewing the segmented tumor region

`TensorFlow` `Streamlit` `OpenCV`
